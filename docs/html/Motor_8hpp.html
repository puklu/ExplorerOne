Detailed Description

Header file for the Motor class, which controls a motor using a PWM and a digital pin.

This file defines the Motor class, which provides functionality to control a motor using a PWM timer for speed control and a digital pin for direction control. It supports operations like moving forward, backward, and halting the motor.

An instance of this class represent one motor with two pins. The speed is controlled using a PWM signal. To simplify the operation, only 1 PWM input is sufficient to drive the motor at desired speed (in percentage of highest speed). The other pin of the motor can simply just be connected to a digital pin.
