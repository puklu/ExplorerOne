Detailed Description

Interrupt Service Routines for GPIO pins.

This file contains implementations of Interrupt Service Routines (ISRs) for handling external interrupts on General Purpose Input/Output (GPIO) pins. The ISRs check for active interrupts on specified pins and invoke the associated callback functions if any are registered.

The functions defined in this file are:

  • ServiceISR0_4: Handles interrupts for EXTI lines 0 to 4.
  • ServiceISR5_15: Handles interrupts for EXTI lines 5 to 15.

Each ISR iterates through the active GPIO pins, checks for pending interrupts, and executes the registered callback functions while ensuring that the interrupts are cleared after servicing.

Note
This file depends on GPIO pin definitions and the interrupt callback mechanism provided in the associated driver files.
