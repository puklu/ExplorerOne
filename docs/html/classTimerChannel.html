Detailed Description

Represents the configurations for an individual channel of a general-purpose timer.

This class serves as a container for the configuration settings needed to set up an individual channel of a general-purpose timer. It stores the necessary parameters that define the behavior and connection of the timer channel, including pin selection, alternate function, capture/compare mode, interrupt settings, and other configuration options.

The TimerChannel class does not perform any actual operations; it simply holds and organizes the configuration data to be passed to the timer during setup.

Note
The actual initialization of GPIO pins and timer settings should be performed separately, typically in a higher-level function that uses this configuration.
