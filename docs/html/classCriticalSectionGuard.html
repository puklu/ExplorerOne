Detailed Description

RAII-style guard for managing critical sections.

This class ensures that interrupts are disabled when the guard is created and re-enabled when the guard goes out of scope. It uses the provided implementation of ICriticalSectionGuard (platform-specific implementation is passed to the ICriticalSectionGuard parameter by whoever wants to use this class) to handle platform-specific interrupt control. It is the responsibility of the caller to pass the desired platform-specific implementation.

Note
This class is non-copyable and non-assignable to prevent misuse.
