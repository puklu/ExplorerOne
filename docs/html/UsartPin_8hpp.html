Detailed Description

Provides the UsartPin class for handling USART pin configurations and data transmission.

The UsartPin class manages USART-specific settings, such as baud rate, word length, parity, and transmission modes, allowing interaction with USART hardware on the specified pin.

◆ ActivateTraceForAssert()

void ActivateTraceForAssert ( )

Configures the system for assertion handling by disabling interrupts and enabling polling-based transmission.

This function disables all USART-related interrupts (RX not empty, TX complete, and TX register empty) to ensure a clean environment during assertion handling. It then redirects the printf output to use polling-based USART transmission for error reporting.
