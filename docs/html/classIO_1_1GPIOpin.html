Detailed Description

Represents a General Purpose Input/Output (GPIO) pin on the microcontroller.

This class provides an interface to configure and control the GPIO pins, including setting the pin direction, enabling pull-up/down resistors, and reading/writing output values. The user can initialize a pin by specifying its identifier, which allows for easy interaction with the hardware.

Example usage:

IO::GPIOpin ledPin(IO::ePin::IO_TEST_LED_LD7);
ledPin.SetResistor(IO::ePupdResistor::IO_RESISTOR_PULL_DOWN);
ledPin.SetDirection(IO::eDirection::IO_DIRECTION_OUTPUT);
ledPin.WriteOutputValue(IO::eValue::IO_VALUE_HIGH);
