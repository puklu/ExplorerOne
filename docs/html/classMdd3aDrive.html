Detailed Description

Implements the IDrive interface for controlling a drive system with four motors using two MDD3A motor drivers. Each driver can drive two motors. So two drivers are used to drive four motors. The input logic for a MDD3A motor driver is:

MDD3A Input logic

IN A IN B MOTOR
0 0 Brake
0 1 Reverse
1 0 Forward
1 1 Brake

This class provides functionality to control a drive system consisting of four motors (front-right, front-left, back-right, and back-left). It supports operations like moving forward, backward, turning left, and turning right, with adjustable speed and turn radius.
