Detailed Description

Manages the active GPIO pin instances.

This header file defines an array of pointers to GpioPin instances, allowing for easy access and management of active GPIO pins within the application. The activePins array is indexed by port and pin numbers, providing a structured way to reference the GPIO pins available in the system.

Note
The activePins array is initialized to nullptr to indicate that no pins are active by default. This array should be used in conjunction with the GpioPin class to manage GPIO functionality across the application.
