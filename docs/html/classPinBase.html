Detailed Description

Abstract base class providing fundamental properties and operations for all pin types.

The PinBase class serves as a foundation for specific pin types, such as GPIO or EXTI pins. It provides core functionality like enabling the pin, setting port and pin numbers, and accessing port and pin details. Derived classes should use this class to extend pin functionality.
