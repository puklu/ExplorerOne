Detailed Description

Defines enumerations for various USART configuration parameters and status flags.

This file contains enumeration types that represent settings for word length, oversampling mode, parity, interrupts, DMA, ISR (Interrupt and Status Register) flags, ICR (Interrupt Clear Register) flags, and baud rates used in configuring and controlling USART functionality.
